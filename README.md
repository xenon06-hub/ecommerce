# 🛒 E-Commerce Application (DevOps Enabled)

A full-stack e-commerce application built with Node.js and deployed using modern DevOps practices including Docker, AWS, Terraform, and CI/CD pipelines.

---

## 🚀 Features

- 🛍️ Product listing and management
- 🧾 Order and cart functionality
- 🔐 Secure backend APIs
- ⚡ Scalable deployment using AWS
- 🔄 CI/CD pipeline automation

---

## 🛠️ Tech Stack

### Backend
- Node.js
- Express.js

### DevOps & Cloud
- Docker (Containerization)
- AWS (ECR, ECS, EC2)
- Terraform (Infrastructure as Code)
- Ansible (Configuration Management)
- Jenkins (CI/CD Pipeline)

### Other Tools
- Git & GitHub
- Linux

---

## ⚙️ DevOps Workflow

1. Developed Node.js application
2. Containerized application using Docker
3. Created Docker image and pushed to AWS ECR
4. Provisioned infrastructure using Terraform:
   - ECS Cluster
   - ECR Repository
   - Load Balancer
5. Configured EC2 using Ansible
6. Deployed app on AWS ECS
7. Automated pipeline using Jenkins
